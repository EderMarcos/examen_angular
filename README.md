# MTech (Examen Angular)

Este proyecto fue generado con [Angular CLI](https://github.com/angular/angular-cli) version 1.6.5. por Jose Eder Marcos Lara

## Para ejecutar el servidor

* Descomprimir el archivo o clonar el repositorio de github
* Tener instalado y actualizado NPM
* Tener instalado y actualizado NodeJS
* Tener instalado y actualizado Angular CLI

Abrir una terminarl dentro de la carpeta y ejecutar primero `npm i` para instalar todos los paquetes necesarios y al terminar, ejecutar `ng serve -o` para que se levante el servidor y se abra el navegador o en su defecto `npm start`.

PD: Este proyecto fue probado en otros 2 equipos de computo y no hubo problema alguno.
